# CyberGuard

CyberGuard is a web-based cybersecurity training platform built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- Interactive phishing simulation
- Password strength analyzer
- Malicious URL detection simulation
- Social engineering simulation
- Dashboard analytics and leaderboard
- Local progress tracking with LocalStorage

## Getting Started

```sh
npm install
npm run dev
```

## Build and Test

```sh
npm run lint
npm test
npm run build
```

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Recharts
- React Router
